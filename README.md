# MDBA-App for drug discovery

- Molecular Descriptors and BioActivity ML App for Drug Discovery.
- This app allows you to generate molecular fingerprints using Molecular Descriptors and using them to predict the 
bioactivity towards inhibting the target enzyme.
- XGBoost Regression is used for predicting the pIC50 values.
- Coefficient of determination achieved is 0.89.
- Acetylcholinesterase is chosen which is a drug target for Alzheimer's disease.
